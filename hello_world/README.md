## Create, compile, and run a simple program

Inside created main.rs, write:

```Rust
fn main() {
    println!("Hello, world!");
}
```

Via. PowerShell (Windows CLI):

```ps
rustc main.rs
```

Two files are created on compilation
- main.pdb
- main.exe

Run main.exe

```ps
.\main.exe
```

Output:

```ps
Hello, world!
```
